Cara Install Debian 11 di Flashdisk


Bagus Koko Wibawanto

24 Juni 2022

Tutorial ini akan membahas tentang bagaimana cara menginstall Debian 11 di Flashdisk. Ini sebenarnya tugas dari sekolah tujuannya nanti Debian nya akan digunakan untuk praktek VoIP (Voice over Internet Protocol).


Ada beberapa yang harus disiapkan untuk menginstall Debian 11 di Flashdisk.


  1. Debian 11
  2. Virtual Box
  3. Flashdisk (16 GB Rekomendasi)

1. Buat VM baru di Virtual Box

Buat VM baru di Virtual Box dan berikan nama (bebas). Untuk spesifikasi VM sebagai berikut:


  • Ram : 1 GB

Pada Langkah berikutnya (Hardisk) ikuti seperti ini:


  • Create a virtual hardisk now
  • Hardisk Type : VDI
  • Dynamical allocated
  • Disk : 16 GB (ini tidak akan dipakai jadi bisa bebas ukurannya).

2. Tambahkan file ISO Debian 11

3. Settingan Tambahan

Pilih menu “USB” untuk memilih flashdisk yang akan digunakan untuk menginstall Debian 11.

4. Jalankan VM dan Install Debian 11

  • BIOS : Install
  • Language : English
  • Region & Location : Asia - Indonesia
  • United State
  • Keyboad : American English
  • Hostname : master (bebas)
  • Domain name : -
  • Root Password : $rootPW
  • User : $userName
  • User Password : $userPW
  • Timezone : Western (Jakarta)

5. Pembagian Partisi

Untuk pembagian partisi pilih "manual" agar bisa bebas menetapkan ukuran. Pada menu "Partition disks" pilih yang Flashdisk.

Buat partisi kosong terlebih dahulu.

Untuk Pembagian Partisi bisa bebas atau bisa ikuti seperti ini:

  • Root (/) : 10 GB
  • Home (/home) : 6 GB

6. Langkah Lanjutan

Ikuti langkah lanjutan seperti ini:

  • Scan Installation Media : No
  • Network Mirror : No
  • Desktop Environment : xfce (bisa selain xfce)

Pada saat install bootloader pilih yang Flashdisk.

Jika sudah selesai, Shutdown VMnya lalu restart Laptop/Komputer dan masuk ke mode BIOS.

Debian 11 selesai diinstall. Untuk kembali ke OS Utama tinggal kembalikan boot menu ke OS utama.
